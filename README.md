# 🛡️ CyberGuardAI

> **Beginner-Friendly Digital Safety Platform For Detecting Scams, Phishing, And Deepfakes**

CyberGuardAI Is A **Privacy-First Security Platform** Designed To Help Everyday Users Stay Safe Online.  
It Detects **Phishing Links, Scam Messages, And Deepfake Images** Using Local Analysis And AI — Without Sending Data To External Services.

---

## ⚠️ Important Notice

> **CyberGuardAI Is NOT A Hacking Tool**

- ❌ No Exploitation Features  
- ❌ No Offensive Capabilities  
- ✅ Built Purely For Protection And Awareness  
- ✅ Designed For Beginners And General Users  

---

## 🧠 Core Features

- 🔍 **Threat Scanner**
  - Analyze Text And URLs For Scam Indicators  

- 🖼️ **Deepfake Image Detection**
  - Lighting Analysis  
  - Texture Inconsistencies  
  - Edge Artifacts Detection  

- 🤖 **Local AI Explanations**
  - Powered By Ollama (`gemma3:1b`)  
  - Human-Readable Risk Analysis  

- 📊 **Risk Scoring System**
  - Clear Verdicts: **Low / Medium / High**  
  - Confidence-Based Results  

- 🎨 **Modern Cyber UI**
  - Black + Red Theme  
  - Smooth Animations  

- 🔐 **Privacy-First Architecture**
  - No External APIs  
  - Fully Local Processing  

---

## 🧱 Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Python (Flask)  
- **AI Engine:** Local LLM Via Ollama  
- **Image Processing:** Pillow + NumPy

## ⚙️ Getting Started

### 🔹 Create Virtual Environment *(Optional)*

```bash
python -m venv venv
venv\Scripts\activate
```

---

### 🔹 Install Dependencies

```bash
pip install -r requirements.txt
```

---

### 🔹 Run The Application

```bash
python app.py
```

---

### 🌐 Open In Browser

```
http://127.0.0.1:5000
```

---

## 🤖 Local AI Setup (Ollama)

CyberGuardAI Uses A Local AI Model For Explanations And Additional Phishing Signals.

### 1. Install Ollama  
### 2. Pull Model

```bash
ollama pull gemma3:1b
```

### 3. Start Ollama

Runs On Default Port:

```
http://localhost:11434
```

---

### 🔹 Optional: Change Model

```bash
set OLLAMA_MODEL=gemma3:1b
```
